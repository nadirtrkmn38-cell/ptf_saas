Sıkça Sorulan Sorular

Planımı istediğim zaman değiştirebilir miyim?

Evet, planınızı istediğiniz zaman yükseltebilir veya düşürebilirsiniz. Yükseltmeler anında aktif olur, düşürmeler mevcut dönem sonunda geçerli olur.

Ödeme yöntemleri nelerdir?

Kredi kartı ve banka kartı ile ödeme yapabilirsiniz. Ödemeler iyzico altyapısı ile güvenli şekilde işlenir.

API limiti aşılırsa ne olur?

Günlük limitinize ulaştığınızda API çağrıları HTTP 429 hatası döndürür. Limit ertesi gün sıfırlanır veya planınızı yükseltebilirsiniz.

İptal politikası nedir?

İstediğiniz zaman iptal edebilirsiniz. İptal ettiğinizde, mevcut ödeme döneminin sonuna kadar hizmetiniz devam eder. Kıst iade yapılmaz.
